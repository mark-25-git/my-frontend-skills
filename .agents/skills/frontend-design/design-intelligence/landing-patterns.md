# Landing Page Structural Patterns

Curated from UIUXPROMAX `landing.csv` (MIT License).
Use this to determine the correct page structure for a given project type.

---

## Pattern Lookup Table

| Pattern Name | Best For | Section Order | CTA Placement | Color Strategy | Conversion Notes |
|---|---|---|---|---|---|
| **Hero + Features + CTA** | SaaS products, software, B2B platforms, tech | 1. Hero (headline/image) → 2. Value prop → 3. Key features (3–5) → 4. CTA → 5. Footer | Hero (sticky) + Bottom | Brand primary for hero. Card bg #FAFAFA for features. Contrasting accent for CTA. | Deep CTA placement. Use contrasting color (7:1 contrast min). Sticky navbar CTA. |
| **Hero + Testimonials + CTA** | Trust-based services, professional services, B2B | 1. Hero → 2. Problem statement → 3. Solution overview → 4. Testimonials → 5. CTA | Hero (sticky) + post-testimonials | Brand color hero. Testimonials on light #F5F5F5 bg. CTA vibrant. | Social proof before CTA. Use 3–5 testimonials with photo + name + role. |
| **Storytelling / Scroll-Driven** | Brand storytelling, creative agencies, premium products | 1. Intro hook → 2. Problem chapter → 3. Journey chapter → 4. Solution chapter → 5. Climax CTA | End of each chapter (mini) + Final CTA | Progressive reveal. Each chapter distinct color. Building intensity. | Narrative increases time-on-page 3×. Use progress indicator. Mobile: simplify animations. |
| **Conversion-Optimized (Funnel)** | Signups, lead gen, SaaS free trials, e-commerce | 1. Hero → 2. Step 1 (problem) → 3. Step 2 (solution) → 4. Step 3 (action) → 5. CTA | Each step: mini-CTA. Final: main CTA | Step 1 red/problem. Step 2 orange/process. Step 3 green/solution. | Progressive disclosure. Form fields ≤ 3. Multiple CTAs at each step. |
| **Social Proof Focused** | Established brands, professional services, e-commerce | 1. Hero → 2. Problem → 3. Social proof (testimonials, logos) → 4. Solution → 5. CTA | After testimonials + Bottom | Trust colors (blue). Success green for results. Neutral review backgrounds. | Show 3–5 testimonials. Real metrics. Star ratings. Verified badges. |
| **Feature-Rich Showcase** | Enterprise SaaS, complex software, multi-feature products | 1. Hero (value prop) → 2. Feature grid/cards (4–6) → 3. Use cases → 4. Social proof → 5. CTA | Hero scroll-trigger + after features + Bottom | Brand primary + card bg #FAFAFA. Feature icons accent. CTA contrasting. | Clear feature hierarchy. One key message per card. Strong CTA repetition. |
| **Minimal & Direct** | Indie products, micro SaaS, freelancers, simple services | 1. Hero headline → 2. Short description → 3. Benefit bullets (3 max) → 4. CTA → 5. Footer | Center, large CTA button | Minimalist: brand + white + accent. Buttons: 7:1+ contrast. Text: black/dark grey. | Single CTA. Large typography. Lots of white space. No nav clutter. Mobile-first. |
| **Trust & Authority** | Legal, finance, healthcare, enterprise, B2B consulting | 1. Hero (mission/credibility) → 2. Proof (logos, certs, stats) → 3. Solution → 4. CTA path | Contact Sales / Get Quote (primary) + nav | Navy/grey corporate. Trust blue. Accent for CTA only. | Security badges. Case studies. Transparent pricing. Low-friction form. |
| **Hero-Centric Design** | Creative agencies, luxury brands, entertainment, hospitality | 1. Full-bleed hero → 2. Single value prop strip → 3. Key benefit or proof → 4. Primary CTA | Hero dominant (center/bottom) + sticky nav CTA | High-impact visual hero. Minimal text. CTA 7:1 contrast. | One primary CTA. Hero is 60–80% above fold. Same hierarchy on mobile. |
| **Interactive Product Demo** | SaaS platforms, dev tools, productivity software | 1. Hero → 2. Product video/mockup (center) → 3. Feature breakdown → 4. Comparison → 5. CTA | Video center + CTA right/bottom | Video surround: brand overlay. Features: icon blue #0080FF. Text: dark #222. | Embedded demo increases engagement. Auto-play video muted. Interactive mockup preferred. |
| **Pricing Page** | SaaS, subscription products, service tiers | 1. Hero (pricing headline) → 2. Pricing cards (3 tiers) → 3. Feature comparison table → 4. FAQ → 5. Final CTA | Each card CTA + sticky nav CTA + bottom | Popular plan: brand color border/bg. Free: grey. Enterprise: dark/premium. | Annual discount 20–30%. Mid-tier 'most popular' badge. FAQs address objections. |
| **Bento Grid Showcase** | Apple-style product pages, feature marketing, SaaS | 1. Hero → 2. Bento Grid (key features) → 3. Detail cards → 4. Tech specs → 5. CTA | Floating action button or bottom of grid | Card bg #F5F5F7 or glass. Icons: vibrant brand colors. Text: dark. | Scannable value props. High information density without clutter. Mobile stack. |
| **App Store Style** | Mobile apps, consumer products with app downloads | 1. Hero with device mockup → 2. Screenshots carousel → 3. Features with icons → 4. Reviews → 5. Download CTAs | Download buttons throughout (App Store + Play Store) | Match app store feel. Star ratings gold. Screenshots with device frames. | Real screenshots. 4.5+ star ratings. QR code for mobile. Platform-specific CTAs. |
| **Waitlist / Coming Soon** | Pre-launch products, beta programs, early access | 1. Hero with countdown → 2. Product teaser → 3. Email capture form → 4. Waitlist count social proof | Email form above fold + sticky on scroll | Anticipation: dark + accent highlights. Countdown brand color. Urgency indicators. | Scarcity + exclusivity. Show waitlist count. Early access benefits. Referral program. |
| **Before-After Transformation** | Beauty, fitness, coaching, any results-driven service | 1. Hero (problem state) → 2. Transformation slider → 3. How it works → 4. Results + CTA | After transformation reveal + bottom | Muted/grey (before) vs vibrant/colorful (after). Success green for results. | Visual proof. 45% higher conversion. Real metrics. Specific results. Guarantee offer. |
| **Portfolio / Gallery** | Creative professionals, photographers, studios, agencies | 1. Hero (name/role) → 2. Project grid (masonry) → 3. About/philosophy → 4. Contact | Project card hover + footer contact | Neutral bg (let work shine). Text: black/white. Minimal accent. | Visuals first. Filter by category. Fast loading critical. |
| **Marketplace / Directory** | P2P platforms, service directories, booking platforms | 1. Hero (search focused) → 2. Categories → 3. Featured listings → 4. Trust/safety → 5. CTA | Hero search bar + nav 'List your item' | Search: high contrast. Categories: visual icons. Trust: blue/green. | Search bar IS the CTA. Reduce friction. Popular searches suggestions. |
| **Enterprise Gateway** | Enterprise software, corporate portals, B2B SaaS | 1. Hero (video/mission) → 2. Solutions by industry → 3. Solutions by role → 4. Client logos → 5. Contact Sales | Contact Sales (primary) + Login (secondary) | Corporate: navy/grey. Conservative accents. High integrity. | Path selection ('I am a…'). Mega menu navigation. Trust signals prominent. |

## Industry → Landing Pattern Quick Map

| Industry | Recommended Pattern |
|---|---|
| SaaS / Software | Hero + Features + CTA or Interactive Product Demo |
| E-commerce (General) | Feature-Rich Showcase or Before-After (if results-driven) |
| E-commerce (Luxury) | Hero-Centric Design or Storytelling |
| B2B Consulting | Trust & Authority or Hero + Testimonials + CTA |
| Healthcare / Medical | Trust & Authority or Social Proof Focused |
| Beauty / Spa | Hero-Centric Design or Social Proof Focused |
| Restaurant / Food | Hero-Centric Design or Minimal & Direct |
| Hotel / Hospitality | Hero-Centric Design or Storytelling |
| Real Estate | Hero + Features + CTA or Bento Grid Showcase |
| Travel / Tourism | Storytelling or Hero-Centric Design |
| Legal / Law Firm | Trust & Authority or Social Proof Focused |
| Finance / Banking | Trust & Authority + Conversion Funnel |
| Creative Agency | Portfolio / Gallery or Storytelling |
| Personal Portfolio | Portfolio / Gallery or Minimal & Direct |
| E-learning / Course | Feature-Rich Showcase + Social Proof |
| Fitness / Gym | Before-After Transformation + Social Proof |
| Wedding / Events | Storytelling or Hero-Centric Design |
| Photography Studio | Portfolio / Gallery or Hero-Centric Design |
| Bakery / Café | Hero-Centric Design or Minimal & Direct |
| AI / Chatbot | Interactive Product Demo or Minimal & Direct |
| Crypto / Web3 | Feature-Rich Showcase or Social Proof Focused |
| Job Board | Marketplace / Directory or Feature-Rich Showcase |
| Pre-launch Product | Waitlist / Coming Soon |
| Enterprise / Corporate | Enterprise Gateway or Trust & Authority |
